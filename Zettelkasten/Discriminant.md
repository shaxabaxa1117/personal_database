# 📏Theme : Discriminant



The main formula for solving quadratic equations is:
$$x_{1,2} = \frac{-b\pm \sqrt{ D }}{2a}$$

$$D=discriminant = b^{2}+4ac$$

```ad-important
if D > 0 => 2 solutions

if D = 0 => 1 solution $$x = -\frac{b}{2a}$$

id D < 0 => no solution
```
